# 🍋 Lemon Squeeze Bot

레몬처럼 상큼하게 🍋  
서버를 더 재밌고 편리하게 만들어주는 디스코드 봇 코드 저장소입니다.

> 🎧 오디오 · 🎨 크리에이터 · 💬 커뮤니티 중심 서버  
> "Lemon Squeeze"를 위한 전용 봇

---

## ✨ Features

- 🎵 **오디오 기능**
  - 음악 재생 / 큐 관리
  - 라디오 스트리밍 지원
  - 음성 채널 자동 관리

- ⚙️ **서버 관리**
  - 유저 경고 시스템
  - 자동 로그 기록
  - 역할 및 권한 관리

- 💬 **커뮤니티 기능**
  - 사연/질문 수집 (라디오용)
  - 이벤트 관리
  - 커맨드 기반 인터랙션

- 🛠️ **유틸리티**
  - 빠른 응답 커맨드
  - 커스텀 메시지
  - API 연동 기능
